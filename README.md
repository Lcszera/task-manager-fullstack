# 📚 Task Manager

De forma objetiva, o Task Manager é uma ideia de projeto que eu tive enquanto estava observando a forma de atribuir processos, agendar salas para reuniões e a visualização de produtos do meu setor. O projeto tem como objetivo:

* Criar tarefas;
* Atribuir tarefas ao usuário;
* Organizar o fluxo de demandas do setor ou da própria rotina do usuário;
* Notificar ao usuário quando o prazo está próximo de ser concluído;

Obs: é possível que o projeto tenha updates referentes à agendar salas para reuniões setoriais.

A aplicação também conta com o registro de alguns dados, com persistência local, guardando credenciais como:

* Login através de e-mail ou nome de usuário;
* Senha do usuário; 
* Foto do usuário (caso seja de sua vontade);
* Tarefas criadas pelo usuário.

Lembrando que, a aplicação também deve realizar cálculos de checagem, para saber quando a garantia do serviço foi excedida. Quando a garantia for excedida, será impressa uma notificação para o usuário ao iniciar a aplicação, informando que o dispositivo cujo id será informado, teve sua garantia excedida.
